# Importa-o-de-dados
Projeto em TypeScript utilizando programação assíncrona para consumir APIs externas (ReceitaWS e ViaCEP). O sistema consulta dados de 7 empresas via CNPJ, busca endereços pelo CEP, cria objetos orientados a objetos, armazena em memória e realiza tratamento de erros com Promise, Async/Await, Resolve e Reject.
